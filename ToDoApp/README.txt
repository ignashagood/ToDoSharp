Приложение для задач, позволяет добавлять, удалять и просматривать задачи

Интерфейс разделен на 2 половины для работы с задачами:

В правой части располагается область добавления задач. Есть поля для установки названия, описания и даты для задачи.
Название не может быть пустым. Задачи с одинаковыми названием и описанием считаются одинаковыми,
дубль задачи нельзя добавить в список. Для добавления задачи вам нужно заполнитьхотя бы поле названия и нажать кнопку "Добавить задачу".
После нажатия кнопки задача отобразится в списке слева.
Также можно добавить задачу из файла. В файле первой строчкой пишется название задачи, второй - описание
и третьей - дата (должна быть описана в форматах "дд.мм.гггг", "дд-мм-гггг", "дд/мм/гггг", "дд\мм\гггг"
и "гггг.мм.дд" формата с такими же разделяющими символами). Для добавления задачи из файла, в нем должна быть хотя бы одна строчка.

В левой части располагается список задач, в верхней части слева можно выбрать тип сортировки (по алфавиту, дате и статусу выполнения),
сортировка применяется только для текущего в момент нажатия списка, при добавлении и удалении элементов сортировка сбивается.
Элемент списка представляет собой строку с делениями. В первой части отображается название задачи, во второй - описание, в третьей - дата,
справа отображается чекбокс для задания статуса выполнения задачи.
Для удаления и просмотра задачи нужно выделить интересующий элемент в списке, после этого в нижней части появятся две кнопки:
"Удалить задачу" и "Открыть задачу".
При нажатии на кнопку удаления, выбранная задача удалится из списка.
При нажатии на кнопку просмотра, откроется новое окно со значениями названия, полного описания, даты и статуса выполнения выбранной задачи. 

Проект сопровождается документацией.

Реализованные требования:
Наличие собственного хранилища сущностей в виде типизированной коллекции (List<Task> Items);
Наличие методов для работы с этим хранилищем (добавление, удаление, сортировка);
Использование интерфейсов при проектировании иерархии классов;
Использование механизма исключений для работы с нештатными ситуациями;
Использование свойств;
Работа с файлами;
